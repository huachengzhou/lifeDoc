### 操作系统的内核

> Yo~生活经验：我们安装完 Windows 操作系统后，会发现操作系统提供了多种多样的功能，比如“记事本”、
  “任务管理器”。然而，这些功能并不是必不可少的。即使没有“任务管理器”，我们仍然可以使用计算机。

![](../.././img/微信截图_20220201172831.png)
![](../../../img/微信截图_20220201172831.png)
