    99、白庙、白猫、白帽

    白庙里有只白猫。

    白庙外有顶白帽，

    白猫看见了白猫，

    叨着白帽跑进了白庙。

    100、瘸子

    北边来了一个瘸子

    背着一捆橛子。

    南边来了一个瘸子

    背着一筐茄子。

    背橛子的瘸子打了背茄子的瘸子一橛子。

    背茄子的瘸子打了背橛子的瘸子一茄子。
