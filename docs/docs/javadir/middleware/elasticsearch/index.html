    elasticsearch与数据库的类比 #

    关系型数据库（比如Mysql） 非关系型数据库（Elasticsearch）
    数据库Database 索引Index
    表Table 类型Type
    数据行Row 文档Document
    数据列Column 字段Field
    约束 Schema 映射Mapping
    • 总结来说elasticsearch字段的含义可以理解为
    • 数据库:Index
    • 表:Type
    • 行:Document
    • 列:Field
    • 约束:Mapping
